# Clinic Management System

A Django-based web application for managing clinic operations, including patient records, prescriptions, medical certificates, and inventory tracking.

## âœ¨ Key Features

- **Patient Management**: Register and manage patient information, including personal details and medical history.
- **Prescription Tracking**: Create, update, and track prescriptions with provider details and timestamps.
- **Medical Certificates**: Generate and manage medical certificates with logbook entries.
- **Inventory Management**: Monitor stock levels for medications and supplies.
- **Referral System**: Handle patient referrals with detailed descriptions and treatment logs.
- **User Authentication**: Secure login and role-based access for providers and administrators.

### Prerequisites
- Python 3.8 or higher
- pip (Python package installer)

### Clone Repository
```bash
git clone https://github.com/KingEddeh/Capstone.git
```

### Navigate to Directory
```bash
cd Capstone
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Set Environment Variables
Create a `.env` file in the root directory and add necessary variables (if any, e.g., for database or secrets).

### Run the Application
```bash
python manage.py runserver
```

Visit `http://127.0.0.1:8000` in your browser.
